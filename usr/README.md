VENTRIX TECHNOLOGIES - Premium Futuristic Startup Website

A cutting-edge Flutter web application showcasing VENTRIX TECHNOLOGIES' expertise in AI, SaaS platforms, cloud systems, and digital transformation. Features a modern dark theme with glowing gradients, smooth animations, background videos, and premium enterprise feel.

## Features

- **Hero Section**: Full-screen cinematic background video with animated gradient text and call-to-action buttons
- **About Section**: Company vision and mission with animated content
- **Services Section**: Grid of 8 service cards with hover animations (AI Solutions, SaaS Development, Mobile Apps, etc.)
- **Technology Stack**: Animated tech icons for React, Next.js, Flutter, Node.js, Python, AWS, Firebase, AI/ML
- **Startup Innovation**: Background video overlay with quote and waitlist CTA
- **Portfolio**: Showcase of tech projects with hover effects
- **Careers**: Section inviting developers, designers, and innovators to join
- **Blog/Insights**: Modern article cards with hover animations
- **Contact/Waitlist**: Glassmorphism contact form with validation
- **Footer**: Links, social media icons, and contact information

## Design Elements

- Dark theme with glowing gradients (blue, purple, cyan)
- Glassmorphism effects with soft neon lighting
- Smooth scrolling animations and fade-in sections
- Micro-interactions on hover
- Floating tech particles background
- Animated counters and interactive UI elements
- Fully responsive design

## Technical Stack

- Flutter for cross-platform development
- Video player for background videos
- Flutter Animate for smooth animations
- Particles Flutter for background effects
- Material Design with custom theming
- SEO optimized for web deployment

## Getting Started

1. Add video assets to `assets/videos/` directory:
   - `hero_background.mp4` (cinematic technology/AI networks video)
   - `startup_innovation.mp4` (futuristic digital world video)

2. Run the app:
   ```bash
   flutter run -d chrome
   ```

3. For production build:
   ```bash
   flutter build web --release
   ```
